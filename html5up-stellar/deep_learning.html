Predicting Student Loan Repayment

with Deep Learning

Objective

Develop two supervised machine learning models (logistic regression and random forest) to accurately classify emails as spam or not spam, enhancing the email filtering system of an Internet Service Provider.

Challenge

The goal was to leverage machine learning to improve the accuracy of spam detection, thereby enhancing the user experience and security. We were to build two models to compare which could identify spam emails with a higher accuracy metric.

Innovation

Employed Pandas for data preprocessing, segmenting the dataset into labels (y) from the 'spam' column and features (X) from the remaining columns. Utilized scikit-learn for developing, training, and testing two machine learning models: logistic regression and random forest. This approach allowed for the comparison of the effectiveness of different algorithms in spam detection.

Process

This project involved several key steps:

  1. Data reading and preprocessing using Pandas.
  2. Splitting data into training and testing sets with scikit-learn's train_test_split.
  3. Feature scaling using scikit-learn's StandarScaler to normalize the feature set.
  4. Building and fitting logistic regression and random forest models using scikit-learn.
  5. Evaluating and comparing the accuracy of both models in classifying spam emails.

Technology Stack

  • Pandas: Utilized for data manipulation and preprocessing, offering robust data analysis capabilities essential for handling the dataset effectively.
  • scikit-learn: Employed for building, training, testing, and evaluating machine learning models, crucial for the logistic regression and random forest classifiers.
  • Python: The primary programming language used for scripting and developing the machine learning models.
  • Jupyter Notebooks: Served as the interactive development environment, facilitating code execution, visualization, and documentation in a comprehensive and user-friendly format.
  • Google Colab: Leveraged for its cloud-based Python programming interface, enhancing collaboration and providing powerful computational resources to run complex machine learning models efficiently.

Outcomes

After developing these two distinct machine learning models it was found that Random Forest Classifier outperformed Logistic Regression in accurately identifying sam emails. This superiority in performance is due to it's ability to handle the complexities of the dataset without overfitting, a common challenge in classification tasks.
